### 第一题

给定一些正整数，将它们连接在一起可以得到另一个数，求最小的这样一个数。

比如，给定

[32, 321] => 32132

[61, 62, 614] => 6146162

### 第二题

求一个完全二叉树的所有路径。树在 ruby 中可以以这样的方式表示：[ [:a], [:b, :c], [:d, :e, :f, g]]（其中 a 是根，b、c 是 a 的两个子节点，d、e 是 b 的子节点，f、g 是 c 的子节点）。

### 第三题

任意一个排序算法的实现，并说明算法复杂度。
